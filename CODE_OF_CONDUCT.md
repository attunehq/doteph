# Code of Conduct

Be respectful and make the project easier for people to participate in.

Do not harass, threaten, demean, or target people with racist, sexist,
homophobic, transphobic, ableist, or otherwise hateful language or behavior.
Good-faith disagreement is fine. Cruelty, intimidation, and bigotry are not.

If you experience or witness a Code of Conduct concern, please report it
privately to the project maintainers rather than raising it in a public issue.
Reports will be handled privately and with care.
